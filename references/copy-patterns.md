# 고전환 카피 패턴

## 헤드라인 공식

### 1. 결과 + 기간 공식
```
[구체적 결과]를 [기간] 만에 달성하는 방법

예시:
- "월 1000만원 매출을 90일 만에 달성하는 시스템"
- "영어 회화 3개월 만에 마스터하는 비밀"
```

### 2. 문제 해결 공식
```
[문제] 없이 [원하는 결과] 얻는 법

예시:
- "광고비 없이 월 500만원 매출 올리는 법"
- "야근 없이 성과 내는 직장인의 비밀"
```

### 3. 타겟 + 혜택 공식
```
[타겟]을 위한 [핵심 혜택]

예시:
- "스마트스토어 초보를 위한 첫 달 100만원 매출 가이드"
- "바쁜 직장맘을 위한 15분 다이어트 프로그램"
```

### 4. 숫자 강조 공식
```
[숫자]명이 선택한 / [숫자]% 성공한

예시:
- "1,247명이 선택한 창업 성공 로드맵"
- "수강생 89%가 3개월 내 성과 달성"
```

### 5. 비교 공식
```
[기존 방법] 대신 [새로운 방법]으로

예시:
- "감에 의존하던 광고, AI가 24시간 최적화합니다"
- "혼자 고민하던 사업, 검증된 시스템으로 시작하세요"
```

## 공감 유발 패턴

### 질문형 오프닝
```
"혹시 이런 경험 있으신가요?"
"이런 고민, 해보셨죠?"
"왜 나만 안 될까... 생각해보신 적 있나요?"
```

### 상황 묘사
```
나쁜 예: "광고가 어려우시죠?"
좋은 예: "매일 밤 12시까지 광고 세팅하고,
        다음 날 ROAS 확인하며 한숨 쉬고 계시진 않나요?"
```

### 감정 훅
```
"저도 그랬습니다"
"혼자가 아닙니다"
"당신 탓이 아닙니다"
```

## CTA 버튼 문구

### 행동 유도형
```
- 지금 시작하기
- 무료로 체험하기
- 바로 신청하기
- 할인 받고 시작하기
```

### 혜택 강조형
```
- 50% 할인 받기
- 보너스 받고 시작하기
- 무료 상담 신청
- 특별 가격으로 시작
```

### 긴급성 유도형
```
- 마감 전 신청하기
- 선착순 마감 임박
- 오늘만 이 가격
- 지금 아니면 정가
```

## 긴급성/희소성 표현

### 시간 한정
```
- "오늘 자정까지만"
- "48시간 한정"
- "이번 주 일요일 마감"
- "D-3 마감 임박"
```

### 수량 한정
```
- "선착순 100명"
- "딱 50명만"
- "현재 23자리 남음"
- "매월 30명 한정"
```

### 보너스 한정
```
- "오늘 신청자에게만 특별 보너스"
- "첫 50명 1:1 컨설팅 무료"
- "지금 가입하면 템플릿 패키지 증정"
```

## 신뢰 구축 표현

### 숫자로 증명
```
- "누적 수강생 2,847명"
- "평균 만족도 4.8/5"
- "재구매율 73%"
- "성공률 89%"
```

### 구체적 결과
```
나쁜 예: "많은 분들이 성공했습니다"
좋은 예: "김OO님, 3개월 만에 월 매출 0원 → 1,200만원"
```

### 권위 차용
```
- "~에서 소개된"
- "~대학 출신"
- "~기업 컨설팅 경험"
- "~년 경력"
```

## 가격 표현

### 가치 비교
```
"총 450만원 상당의 구성을
오늘만 49만원에"
```

```
"하루 커피 한 잔 값으로
평생 사용하는 시스템"

"하루 1,600원으로
월 1000만원 매출 시스템 구축"
```

### 투자 대비 수익
```
"49만원 투자로
첫 달 100만원 매출 가능"

"이 금액은 첫 고객 한 명이면 회수됩니다"
```
